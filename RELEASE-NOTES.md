# Release Notes

## v0.4.0

**Dynamic noise for extreme motion.** Micro avatars now flicker with Wall Street rush effect - random character noise mimics high-intensity activity. Shifting gradient patterns, hybrid breathing, protected frame slots. Idle expressions API added. 70% of vision achieved after multiple experiments.

Documentation restructured: `decisions/` with 5 searchable ADRs, CLAUDE.md guide, minimal changelog.

## v0.3.0

**TangentClient API & Compact Micro Avatars.** Framework-agnostic animation controller with thread-safe state management. Per-state FPS, state aliases, auto-tick, queuing. Micro avatars now 8x2 (down from 10x2). 98% reduction in integration code.

**Ecosystem Integration & Code Generation.** Frame cache API for O(1) access (500x faster). Bubble Tea component for plug-and-play integration. Code generation from constants.go as single source of truth.
